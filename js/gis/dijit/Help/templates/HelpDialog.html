Navegación utilizando teclado y mause:
  • Arrastre con el mause para moverse
  • SHIFT + Click para volver al centro.
  • SHIFT + Arrastrar para acercar.
  • SHIFT + CTRL + Arrastrar para alejar.
  • Gira la rueda al frente para acercar.
  • Gira la rueda hacia atrás para alejar.
  • Usa las flechas para mover el mapa.
  • Presiona + para acercar un nivel.
  • Presiona - para alejar un nivel.
  • Doble click para acercar y centrar.
In addition to Search capabilities, the following tools are provided:
  • Measure
    • The measure tool provides the capabilities to draw a point, line, or polygon on the map and specify the unit of measurement.
  • Print
    • This map can be exported to varouis formats and layouts
